Установка

  • Отсюда качаем Mysaifu JVM и Java Communications API для него, на сегодня версии 0.4.7 и 0.0.3, соответственно.
  • jvm.0.4.7-bin.zip распаковываем и копируем CAB на КПК, там его запускаем и устанавливаем, можно на карточку.
  • javaxcomm-0.0.3-bin.zip распаковываем в устаночную директорию, обычно это "\Карта памяти\Program Files\Mysaifu JVM"
  • Распаковываем sasplanetj.zip и переписыаем sasplanetj на КПК, например в "\Карта памяти\sasplanetj"
  • Создаём там же папку "cache", переписываем туда нужный кеш с компьютера. Как лучше всего это сделать - смотрите ниже на сайте.
  • Теперь из "Программ" запускаем Mysaifu и указываем параметры для запуска sasplanetj
  • В CLASSPATH на надо указать путь к папке "\Карта памяти\sasplanetj\sasplanetj.jar" и к "\Карта памяти\Program Files\Mysaifu JVM\jre\lib\javaxcomm.jar"
    т.е. CLASSPATH=\Карта памяти\sasplanetj\sasplanetj.jar;\Карта памяти\Program Files\Mysaifu JVM\jre\lib\javaxcomm.jar
  • Опционально можно ещё указать Current directory: текущая директория, где лежит кеш, конфиги, программа и т.д. "\Карта памяти\sasplanetj"
  • Type: Class
  • Name: org.sf.sasplanetj.App (какой класс запускать)
  • Command line arguments:
    • без параметров запустится симуляция из nmealog.txt (в котором должны быть NMEA сентенции)
    • list - выведет список доступных портов
    • <port> [<baudrate>] - порт на котором находится GPS приёмник, например "COM4:", скорость (необязательно)
  • В опциях можно указать больше памяти доступной для Java, по-умолчанию всего 2Мб.
  • Ставим галочку "Show console", чтобы видеть консоль с отладочной информацией (необязательно)


Нажимаем "Execute"

Hot keys (Клавиши управления)

  • PgUp, F1, Shift+Up - zoom in;
  • PgDn, F2, Shift+Down - zoom out;
  • Left/Right/Up/Down - navigate;
  • Enter - center map (return to origin);
  • '1', Shift+Right - Switch to next map;
  • '2', Shift+Left - Switch to previous map.

Supported maps (cache folders)

  • Google satellite (SAT);
  • Google map (MAP);
  • Google landscape (land);
  • Yandex satellite (yasat);
  • Yandex map (yamapng);
  • OpenStreetMap (osmmap);
  • Virtual Earth satellite (vesat);
  • Gurtam (gumap);
  • WikiMap (WikiMap).

Supported layer: Wiki KML.
