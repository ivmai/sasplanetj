Установка

  • Отсюда качаем Mysaifu JVM и Java Communications API для него, на сегодня версии 0.4.5 и 0.0.3.
  • jvm.0.4.5-bin.zip распаковываем и копируем CAB на КПК, там его запускаем и устанавливаем, можно на карточку.
  • javaxcomm-0.0.3-bin.zip распаковываем в устаночную директорию, обычно это "\Карта памяти\Program Files\Mysaifu JVM"
  • Распаковываем sasplanetj.zip и переписыаем sasplanetj на КПК, например в  "\Карта памяти\sasplanetj"
  • Создаём там же папку "cache", переписываем туда нужный кеш с компьюра. Как лучше всего это сделать - смотрите ниже на сайте.
  • Теперь из "Программ" запускаем Mysaifu и указваем параметры для запуска sasplanetj
  • В CLASSPATH на надо указать путь к папке "\Карта памяти\sasplanetj\bin" и к "\Карта памяти\Program Files\Mysaifu JVM\jre\lib\javaxcomm.jar"
    т.е. CLASSPATH=\Карта памяти\sasplanetj\bin;\Карта памяти\Program Files\Mysaifu JVM\jre\lib\javaxcomm.jar
  • Current directory: текущая директория, где лежит кеш, конфиги, программа и т.д. "\Карта памяти\sasplanetj"
  • Type: Class
  • Name: sasplanetj.App (какой класс запускать)
  • Command line arguments:
    • без параметров запустится симуляция из nmealog.txt (в котором должны быть NMEA сентенции)
    • list - выведет список доступных портов
    • <port> [<baudrate>] - порт на котором находится GPS приёмник, например "COM4:", скорость (необязательно)
  • В опциях можно указать больше памяти доступной для Java, по-умолчанию всего 2Мб.
  • Ставим галочку "Show console", чтобы видеть консоль с отладочной информацией
